# 🚀 Fake News Detection System using Hadoop & Node.js

## 📌 Description
This project detects fake news using Big Data technologies like Hadoop and Node.js.

## 🧠 Technologies Used
- Hadoop (HDFS + MapReduce)
- Node.js (Express)
- Python (Mapper/Reducer)
- MySQL
- HTML/CSS/JS

## ⚙️ Features
- Real-time fake news detection
- Hadoop-based processing
- Word frequency analysis
- REST API integration

## ▶️ How to Run

### Start Hadoop
start-dfs.sh
start-yarn.sh

### Run Backend
cd backend
node server.js

### Open Frontend
Open frontend/index.html in browser

## 📊 Sample Output
REAL / FAKE prediction with word count
